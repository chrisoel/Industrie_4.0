# industrie_4.0

Entwicklung eines praxisorientierten Industrie-4.0-Kurses für Techniker.
Ziel: zentrale Technologien und Konzepte mit hohem Praxisanteil vermitteln.

## Projektstruktur

- `python/` enthält den vollständigen Python-Lernpfad mit nummerierten Notebooks.
- Weitere Teilbereiche können als eigene Unterordner ergänzt werden.

## Produktion

- Inhalte sind für den Einsatz in Kursumgebungen vorbereitet.
- Für Ausführung der Python-Notebooks die Anleitung in `python/README.md` nutzen.
- Abhängigkeiten in isolierten Umgebungen installieren (venv).
